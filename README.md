# GPT to Tripian

<img src="./ss.jpg">

### Install

Configure API keys in `.env`:

```
GPT_API_KEY=xxx
```

`constants-example.js` to `constants.js`:

```js
export const param = Object.freeze({
    GOOGLE_MAP_KEY: 'GOOGLE_MAP_KEY',
});
```

Install dependencies:

```shell
npm install
```

### Run

```shell
npm run dev
```
